## Getting Started

1. Clone the repo:

```bash
git clone https://github.com/your-username/midjourney-prompt-library.git
cd midjourney-prompt-library
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm run dev
```

## Tech Stack

* Next.js
* Tailwind CSS
* MongoDB
* Ky (API requests)
